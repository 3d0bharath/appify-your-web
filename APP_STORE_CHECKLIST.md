Use this checklist to make sure you have everything ready before submitting to app stores.

## ü§ñ Google Play Store Checklist

### Developer Account
- [ ] Google Play Console account created ($25 paid)
- [ ] Payment profile set up
- [ ] Tax information completed (if selling paid app/IAP)

### App Assets
- [ ] 512x512px app icon (PNG, no transparency)
- [ ] 1024x500px feature graphic
- [ ] Phone screenshots (1080x1920px or 1080x2340px, minimum 2, max 8)
- [ ] 7-inch tablet screenshots (optional but recommended)
- [ ] 10-inch tablet screenshots (optional but recommended)
- [ ] Short description (80 characters max)
- [ ] Full description (4000 characters max)

### App Configuration
- [ ] App name finalized
- [ ] Category selected (Health & Fitness)
- [ ] Tags added (wellness, yoga, diet, lifestyle)
- [ ] Content rating questionnaire completed
- [ ] Target audience & content completed
- [ ] Data safety form filled out
- [ ] Store listing contact details added
- [ ] Privacy policy URL added
- [ ] App APK/AAB built and signed

### Pre-Launch
- [ ] Internal testing completed (optional but recommended)
- [ ] Closed testing with beta users (optional)
- [ ] All Play Console sections have green checkmarks
- [ ] Release notes written

### Post-Submission
- [ ] Monitor Play Console for review status
- [ ] Respond to any review questions within 7 days
- [ ] Check for approval email
- [ ] Test live app after approval

**Expected Review Time:** 1-7 days

- [ ] App name finalized (max 30 characters)

- [ ] Content warnings reviewed

- [ ] iOS app archived in Xcode
- [ ] Build uploaded to App Store Connect
- [ ] Build selected in App Store Connect
- [ ] Test account credentials provided (if app requires login)

### Post-Submission

- [ ] Respond to any reviewer questions quickly

- [ ] Test live app after approval
